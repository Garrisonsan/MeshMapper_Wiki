# 🚀 MeshMapper Update (February 25, 2026)

### ✨ New Features

* **New Admin Panels** — Completely redesigned admin panel UI - and now with mobile support!

![Image](https://meshmapper.net/img/upload_20260225_174314_06fb3bb6.jpg)

* **Repeater Neighbours** — When selected on a repeater, clicking another repeater under "Neighbours" will display the neighbour on the map and draw neighbour lines.

![Image](https://meshmapper.net/img/upload_20260225_174525_84135693.jpg)

* **Leaderboard Links** — Clicking on a repeater on a regional or global leaderboard will open a new tab and take you directly to that repeater. (Thanks Vgo)

* **Neighbour Boundries** — A new "Neighbour Zone Boundries" map layer will expose neighbouring regions boundries. (Thanks davidjay)

![Image](https://meshmapper.net/img/upload_20260225_174958_262c7a9c.jpg)

* **Neighbour Boundries** — Regional administrators can now see neighbouring regions boundries when editing their own from within the admin panel.

![Image](https://meshmapper.net/img/upload_20260225_175102_3a619ffd.jpg)

* **Duplicate Repeater Cleanup** — Instead of a hardcoded 3 days, repeaters in active ID collision will delete themselves after 3x the number of hours set as the regions Stale Repeater Age. (Thanks to many)
